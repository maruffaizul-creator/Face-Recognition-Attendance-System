Face Recognition Attendance System

A smart attendance system developed using Python, OpenCV, K-Nearest Neighbors (KNN), and Streamlit. The system detects and recognizes faces through a webcam, marks attendance automatically, and stores records in CSV files. The project uses facial image data for training and provides a simple dashboard to view attendance records.

Technologies Used

- Python
- OpenCV
- NumPy
- Pandas
- Scikit-Learn (KNN)
- Streamlit

Features

- Face Detection
- Face Recognition
- Automated Attendance Marking
- Attendance Dashboard
